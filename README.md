# 2D-Physics-Simulation
This Repository contains my take on simulating some physics concepts I learnt from the high-school curriculum and first year university.

## Purpose
Back in high-school (around 2023), I took physics as one of my compulsory subjects, and I found it really fascinating. However, I also found that it could be hard to understand some of the concepts without a visual aid. I decided to make small Python (PyGame) and C++ (SFML) simulations to help me visualize these concepts, these can be found in the [`/oldSims`](/oldSims) folder.

Later on, around 2025, I decided to remake them into a single, cohesive website using Vanilla JavaScript and the `<canvas>` element with a self-implemented physics engine. I plan to add upon it in the future for improvement.

## Physics Concepts Simulated
- Friction & Gravity
- Elastic and Inelastic collisions
- Projectile Motion
- Cathode Ray Tubes (CRT)
- Diffraction Grating
